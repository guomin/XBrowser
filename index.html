XBrowser by axefrog

A fully-managed "headless" web browser for .Net

This is an early work in progress. The intention is to have a fully-functional HTML5 W3C-compliant browser with full JavaScript support. The goal will be to be able to successfully load and browse any standard non-Flash website using scriptable automation rather than a rendered user interface.

Dependencies

XBrowser requires an installation of the .Net framework 4.
